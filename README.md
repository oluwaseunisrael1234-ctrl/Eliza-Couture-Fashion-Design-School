Fashion Designer Website

Project Overview
This project is a modern, responsive website for a fashion designer who offers bespoke clothing services and runs a fashion school. The website is designed to showcase sewing services, fashion courses, portfolios, and client testimonials. It also includes a seamless booking system that allows clients and students to schedule appointments via WhatsApp.
The design focuses on a luxury aesthetic, smooth animations, and mobile-first responsiveness from 300px upward, ensuring usability across all devices.

Features
* **Responsive Design:** Fully optimized for all devices, from mobile phones (300px) to large desktops.
* **Hero & Portfolio Sections:** Eye-catching hero banner, services overview, and portfolio/gallery to highlight fashion work.
* **Fashion School Integration:** Information about courses, core values, and school offerings.
* **Booking System:** Clients and students can book appointments via WhatsApp seamlessly.
* **Testimonials:** Showcases client and student feedback for credibility.
* **Call-to-Action Sections:** Encourages engagement and appointment booking.
* **Contact Page:** Integrated contact form and Google Maps for location details.
* **Smooth Animations:** Fade-in, hover effects, and scale transitions for a professional feel.
* **Lightweight & Clean Code:** Mobile-first CSS with Flexbox and Grid layouts.

Technologies Used
* **HTML5** – Semantic markup for structure
* **CSS3** – Custom styling, animations, and responsive media queries
* **JavaScript (Optional)** – For interactive elements and gallery lightbox
* **Flexbox & Grid** – For layout management
* **WhatsApp API/Link Integration** – For booking appointments
